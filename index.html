Sobre Mí

Soy Alexis Jara, especialista en diseño gráfico, desarrollo web y red team, apasionado por la tecnología y la creatividad. Mi camino como autodidacta me ha permitido combinar la seguridad informática con el desarrollo de proyectos innovadores, siempre orientado a resultados y con la convicción de que el aprendizaje constante es la clave del progreso.

He construido una comunidad de más de 19.9k seguidores en TikTok, donde comparto contenido de hacking ético para educar e inspirar a otros. Además, lidero dos emprendimientos que reflejan mi visión profesional: ZyLogix, una tienda de productos electrónicos, y Kreatix (que pronto evolucionará a Alnex Studio), una agencia que integra diseño gráfico, desarrollo web y aplicaciones web. Mi propósito es mostrar profesionalismo en cada proyecto y, al mismo tiempo, motivar a otros a creer en su potencial y transformar ideas en realidad.

Lenguajes

  1. Python

    Utilidad Mía: Scripts

    Python es un lenguaje de programación versátil, de alto nivel y de código abierto, conocido por su sencillez y legibilidad, lo que lo hace ideal para principiantes y profesionales. Sirve para una amplia gama de aplicaciones, incluyendo desarrollo web (principalmente el backend), ciencia de datos, inteligencia artificial (IA), machine learning (ML), y la automatización de tareas.

  2. Dart

    Utilidad Mía: Desarrollo Móvil

    Dart es un lenguaje de programación moderno, de código abierto y orientado a objetos, desarrollado por Google con el objetivo de ser un lenguaje eficiente para el desarrollo de aplicaciones rápidas en cualquier plataforma. Su versatilidad lo hace ideal para crear aplicaciones web, móviles, de escritorio y del lado del servidor, especialmente conocido por ser el lenguaje detrás del framework Flutter para desarrollo multiplataforma.

  3. HTML, CSS, JavaScript

    Utilidad Mía: Desarrollo Web

    HTML, CSS y JavaScript son los tres lenguajes fundamentales para crear el front-end (la parte que el usuario ve) de un sitio web. HTML (Lenguaje de Marcado de Hipertexto) define la estructura y el contenido de la página web; CSS (Hojas de Estilo en Cascada) se encarga del diseño y la apariencia visual (colores, fuentes, etc.); y JavaScript añade interactividad y dinamismo al sitio, permitiendo que los elementos se muevan y cambien
